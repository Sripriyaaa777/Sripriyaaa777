### 🚀 Featured Projects

**🛡️ [GigShield — Parametric Insurance Platform for Gig Workers](https://github.com/Sripriyaaa777/DEVTrails)** `🥈 2nd/18,200+ · Guidewire DevTrails Hackathon 2026`
Full-stack AI-integrated platform that auto-triggers insurance payouts for gig workers using real-time disruption signals (weather, AQI, traffic). Built an XGBoost + Prophet risk-pricing engine and an Isolation Forest fraud-detection layer, on React PWA + Node.js + FastAPI + PostgreSQL + Redis + Docker.

**🕸️ [RiskTrace Engine — Multi-Agent AI for Cascading Risk Analysis](https://github.com/Sripriyaaa777/RiskTrace-Engine)** `Agentic AI`
6-agent system (Perception → Graph Reasoning → Planning → Decision → Monitoring → Critic) that traces cascading risk across temporal dependency graphs from issue-tracker data, on a Neo4j backend. Integrates Groq LLMs for root-cause explanation and a counterfactual what-if engine for real-time risk recomputation.

**🚁 [GenAI-Powered Rescue Drone System](https://github.com/Sripriyaaa777/GenAI-Powered-Rescue-Drone-System)** `🏅 Finalist · CMR HackFest 2.0 2025`
Autonomous search & rescue drone concept combining computer vision, predictive risk mapping, and real-time decision-making for disaster response.

**🆘 [HelpMeNow — AI-Powered Women's Safety Platform](https://github.com/Sripriyaaa777/HelpMeNow)** `🥇 Winner · Epochon Hackathon 2024`
SOS alerts, live tracking, and ML-based threat detection built on Python, Firebase, and Twilio.

**💊 [Siamese GNN for Adverse Drug Event Prediction](https://github.com/Sripriyaaa777/Drug-Drug-interaction)** `Graph ML`
Converts DrugBank SMILES strings into molecular graphs (RDKit) and predicts drug-drug interaction probability via a Siamese GNN + regularized MLP, with drug-level splitting and negative sampling to prevent data leakage.

**🗣️ [Malayalam-English Code-Mixed LLM Benchmarking](https://github.com/Sripriyaaa777/malayalam-llm-benchmark-research)** `📄 Under Review 2025`
Benchmarked Llama 3.3 70B, Mistral Large, and Gemma 2 9B on 9K+ code-mixed samples (FIRE 2020), surfacing severe script-handling failures (up to 100% invalid outputs) and proposing *valid prediction rate* as a new LLM reliability metric. Zero/few-shot prompting pipelines improved accuracy by ~23%.
↳ related: [malayalam-llm-benchmark](https://github.com/Sripriyaaa777/malayalam-llm-benchmark)

**⚛️ [Qiskit Workshop — Hybrid Quantum-Classical NN](https://github.com/Sripriyaaa777/Qiskit_workshop)** `IBM Quantum · Qiskit Fall Fest 2025`
Built a QNN with ZZFeatureMap and RealAmplitudes ansatz (optimized via COBYLA) to classify the Two Moons dataset — hands-on exposure to quantum ML and VQE.
